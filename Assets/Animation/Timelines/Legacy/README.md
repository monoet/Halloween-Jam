# Animation Timelines

Este folder contiene los assets utilizados por el sistema JRPG Animation System.

- ActionTimelineCatalog.asset: catalogo de timelines por id de accion. Usa el menu Battle/Animation/Validate Action Timeline Catalog para validar y puedes ejecutar Rebuild Lookup desde el inspector si cambias la lista.
- asic_attack.asset y magic_bolt.asset: ejemplos demo para las pruebas iniciales.

No confundir con el ActionCatalog existente en Assets/BattleConfig, que corresponde al catalogo de acciones de gameplay.
