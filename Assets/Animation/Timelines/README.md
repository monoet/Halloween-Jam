# Animation Timelines

Este folder contiene los assets utilizados por el sistema JRPG Animation System.

- ActionTimelineCatalog.asset: catálogo de timelines por id de acción.
- asic_attack.asset y magic_bolt.asset: ejemplos demo.

No confundir con el ActionCatalog existente en Assets/BattleConfig, que corresponde al catálogo de acciones de gameplay.
